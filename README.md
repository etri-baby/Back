## 1️⃣ 프로젝트 개요

> 다양한 IoT 디바이스를 효율적으로 관리하고 제어하기 위한 통합 플랫폼을 개발
> 

### Backend Technology

| 도구 명 | 설명 |
| --- | --- |
| Java(JDK17) | 개발 언어 |
| Gradle | 빌드 도구 |
| Spring Boot(v3.1.1) | 서버 프레임워크 |
| MariaDB(v10.5) | 관계형 데이터베이스 |
| Swagger3 Docs | API 문서 도구 |
| Docker | 컨테이너 도구 |
| MQTT Client | IoT 통신 프로토콜 |
